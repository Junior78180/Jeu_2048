# Jeu_2048
Ce projet est une version console du jeu **2048**, développée en langage **C**.   Le but du jeu est simple : combiner des tuiles portant des puissances de 2, jusqu'à atteindre la tuile **2048**.   Il se joue dans une grille de 4x4 cases, et chaque mouvement peut faire apparaître une nouvelle tuile.
